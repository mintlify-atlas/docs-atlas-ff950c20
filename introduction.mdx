## What is Gentle AI?

Gentle AI is **NOT** an AI agent installer. Most agents are easy to install. This is an **ecosystem configurator** — it takes whatever AI coding agent(s) you use and supercharges them with the Gentleman stack.

## The Ecosystem Stack

Gentle AI configures your agents with these components:

- **Engram** — Persistent cross-session memory that survives context resets
- **SDD** — Spec-Driven Development workflow with 9 structured phases
- **Skills** — Curated coding patterns including SDD workflow skills and foundation skills like Go testing
- **Context7** — MCP server for real-time library and framework documentation
- **GGA** — Gentleman Guardian Angel AI provider switcher for multi-model workflows
- **Persona** — Security-first permissions and teaching-oriented behavior
- **Theme** — Gentleman Kanagawa theme overlay

## Key Features

### One Command Installation

No manual configuration needed. The CLI and TUI handle everything:

```bash
gentle-ai
```

### Cross-Platform Support

Works on macOS, Linux (Ubuntu/Debian/Arch), and Windows 10/11 with intelligent package manager detection.

### Preset-Based Configuration

Choose from curated presets or build your own:

- **Full Gentleman**: All components, all skills, gentleman persona
- **Ecosystem Only**: Core components, P0 skills, gentleman persona
- **Minimal**: Engram + Persona + Permissions only
- **Custom**: Pick exactly what you need

### Safe and Reversible

Automatic backups before every change with one-command restore.
