## What Is Gentle AI?

This is **NOT** an AI agent installer. Most agents are easy to install. This is an **ecosystem configurator** — it takes whatever AI coding agent(s) you use and supercharges them with the Gentleman stack.

## Quick Install

<CodeGroup>

```bash macOS / Linux
curl -fsSL https://raw.githubusercontent.com/Gentleman-Programming/gentle-ai/main/scripts/install.sh | bash
```

```powershell Windows
irm https://raw.githubusercontent.com/Gentleman-Programming/gentle-ai/main/scripts/install.ps1 | iex
```

```bash Homebrew
brew tap Gentleman-Programming/homebrew-tap
brew install gentle-ai
```

```bash Go Install
go install github.com/gentleman-programming/gentle-ai/cmd/gentle-ai@latest
```

</CodeGroup>

This downloads the latest release for your platform and launches the interactive TUI. No Go toolchain required for the shell installers.
