## Overview

This quickstart guide will have you configuring your AI coding agents with the Gentle AI ecosystem in under 5 minutes.

## Prerequisites

- **macOS**, **Linux** (Ubuntu/Debian/Arch), or **Windows 10/11**
- One or more AI coding agents installed:
  - [Claude Code](https://claude.ai/download)
  - [OpenCode](https://opencode.sh)
  - [Gemini CLI](https://ai.google.dev/gemini-api/docs/cli)
  - [Cursor](https://cursor.sh)
  - [VS Code](https://code.visualstudio.com) with GitHub Copilot

<Note>
Don't have an agent installed yet? No problem! Gentle AI can configure any agent after you install it.
</Note>

## Installation

Choose your installation method:

<CodeGroup>

```bash macOS / Linux
curl -fsSL https://raw.githubusercontent.com/Gentleman-Programming/gentle-ai/main/scripts/install.sh | bash
```

```powershell Windows
irm https://raw.githubusercontent.com/Gentleman-Programming/gentle-ai/main/scripts/install.ps1 | iex
```

```bash Homebrew
brew tap Gentleman-Programming/homebrew-tap
brew install gentle-ai
```

```bash Go Install
go install github.com/gentleman-programming/gentle-ai/cmd/gentle-ai@latest
```

</CodeGroup>

This downloads the latest release for your platform and launches the interactive TUI.

## Interactive Configuration

<Steps>
  <Step title="Select Your Agents">
    Choose which agents to configure:
    
    ```
    ┌─ Select Agents ──────────────────────┐
    │ [x] Claude Code                      │
    │ [ ] OpenCode                         │
    │ [ ] Gemini CLI                       │
    │ [ ] Cursor                           │
    │ [ ] VS Code Copilot                  │
    └──────────────────────────────────────┘
    ```
    
    Use arrow keys to navigate, space to select, enter to continue.
  </Step>
  
  <Step title="Choose a Preset">
    Select your configuration level:
    
    - **Full Gentleman**: All components + all skills + gentleman persona
    - **Ecosystem Only**: All components + core skills + gentleman persona
    - **Minimal**: Engram + Persona + Permissions only
    - **Custom**: Pick components and skills individually
    
    **Recommendation**: Start with **Ecosystem Only** for a balanced setup.
  </Step>
  
  <Step title="Review & Install">
    Gentle AI will:
    
    1. Detect missing dependencies (git, curl, node, npm)
    2. Show the installation plan
    3. Create a backup of existing configs
    4. Install selected components
    5. Verify installation success
    
    The entire process takes 30-60 seconds.
  </Step>
  
  <Step title="Verify Installation">
    After installation completes, you'll see a verification report:
    
    ```
    ✓ Claude Code configured successfully
    ✓ Engram protocol injected
    ✓ SDD orchestrator installed
    ✓ 11 skills copied to ~/.claude/skills/
    ✓ Context7 MCP server configured
    ✓ Gentleman persona applied
    ```
  </Step>
</Steps>

## Verify It Works

<Tabs>
  <Tab title="Claude Code">
    Open Claude Code and try:
    
    ```
    /sdd-init
    ```
    
    This initializes SDD context in your current project. Claude will ask about your project architecture and save it to Engram memory.
  </Tab>
  
  <Tab title="OpenCode">
    Open OpenCode and try:
    
    ```
    /sdd-init
    ```
    
    The agent will bootstrap SDD context for your project using the installed skills.
  </Tab>
  
  <Tab title="Gemini CLI">
    Run:
    
    ```bash
    gemini "Help me initialize SDD for this project"
    ```
    
    Gemini CLI will use the `sdd-init` skill to set up project context.
  </Tab>
  
  <Tab title="Cursor">
    In Cursor's chat:
    
    ```
    Initialize SDD context for this project
    ```
    
    Cursor will detect the SDD skills and guide you through initialization.
  </Tab>
  
  <Tab title="VS Code Copilot">
    In VS Code chat:
    
    ```
    @workspace Initialize SDD context
    ```
    
    Copilot will use the installed skills to bootstrap SDD for your workspace.
  </Tab>
</Tabs>

## Troubleshooting

<Accordion title="Dependencies Missing">
Gentle AI auto-detects missing tools (git, curl, node, npm) and provides platform-specific installation commands.

**macOS**: `brew install <tool>`  
**Linux**: `apt install <tool>` or `pacman -S <tool>`  
**Windows**: `winget install <tool>`
</Accordion>

<Accordion title="Permission Errors">
If npm global install fails with permission errors:

```bash
# Option 1: Use a node version manager (recommended)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install node

# Option 2: Fix npm prefix
npm config set prefix ~/.npm-global
export PATH=~/.npm-global/bin:$PATH
```
</Accordion>

<Accordion title="Agent Not Detected">
If Gentle AI doesn't detect your installed agent:

1. Check the agent is installed at the expected path
2. Manually create the config directory if needed
3. Run `gentle-ai install --agent <agent-id>` explicitly
</Accordion>
