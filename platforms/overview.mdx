## Platform Detection

Gentle AI automatically detects your environment:

### Operating System

```go
// internal/system/detect.go:34
runtime.GOOS  // darwin, linux, windows
runtime.GOARCH // amd64, arm64
```

<Tabs>
  <Tab title="macOS">
    **Detection**:
    - OS: `darwin`
    - Arch: `amd64` (Intel) or `arm64` (Apple Silicon)
    - Package Manager: Homebrew (default)
  </Tab>
  
  <Tab title="Linux">
    **Detection**:
    - OS: `linux`
    - Distro: Parsed from `/etc/os-release`
      - Ubuntu (ID=ubuntu or ID_LIKE contains ubuntu)
      - Debian (ID=debian or ID_LIKE contains debian)
      - Arch (ID=arch or ID_LIKE contains arch)
    - Package Manager: apt, pacman, or brew (if installed)
  </Tab>
  
  <Tab title="Windows">
    **Detection**:
    - OS: `windows`
    - Arch: `amd64` or `arm64`
    - Package Manager: winget (default)
    - Shell: PowerShell (when $SHELL not set)
  </Tab>
</Tabs>

### Package Managers

Gentle AI detects and uses your system's package manager:

| Platform | Default | Detected | Install Command |
|----------|---------|----------|----------------|
| macOS | Homebrew | `brew --version` | `brew install` |
| Ubuntu/Debian | apt | `apt --version` | `sudo apt install` |
| Arch Linux | pacman | `pacman --version` | `sudo pacman -S` |
| Windows | winget | `winget --version` | `winget install` |

<Info>
On Linux, if Homebrew is installed, Gentle AI prefers it over apt/pacman for better cross-platform consistency.
</Info>

### Dependency Detection

Gentle AI checks for required tools:

```bash
# Tools checked
git --version
curl --version
node --version
npm --version
```

**Result**:
- `Installed`: Tool found and version detected
- `Not Found`: Missing dependency
- `InstallHint`: Platform-specific command to install

## Configuration Paths

Agent configuration paths vary by platform:

### macOS / Linux

```bash
~/.claude/                    # Claude Code
~/.config/opencode/           # OpenCode
~/.gemini/                    # Gemini CLI
~/.cursor/                    # Cursor
~/.copilot/                   # VS Code Copilot (skills)
~/Library/Application Support/Code/User/   # VS Code (macOS)
~/.config/Code/User/          # VS Code (Linux)
```

### Windows

```powershell
%USERPROFILE%\.claude\                      # Claude Code
%USERPROFILE%\.config\opencode\             # OpenCode
%USERPROFILE%\.gemini\                      # Gemini CLI
%USERPROFILE%\.cursor\                      # Cursor
%USERPROFILE%\.copilot\                     # VS Code Copilot (skills)
%APPDATA%\Code\User\                        # VS Code
```

<Accordion title="Path Resolution">
Gentle AI uses Go's `os.UserHomeDir()` to resolve the home directory:

- **macOS/Linux**: `$HOME`
- **Windows**: `%USERPROFILE%`

All paths are expanded from this base.
</Accordion>

## Platform-Specific Behavior

### npm Global Installs

<Tabs>
  <Tab title="macOS / Linux">
    **Default**: Requires `sudo` for system-wide npm prefix.
    
    **Detection**: Gentle AI checks if npm prefix is user-writable:
    ```bash
    npm config get prefix
    # If starts with $HOME → user-writable (no sudo needed)
    # If /usr/local or /usr → requires sudo
    ```
    
    **Recommendation**: Use a node version manager (nvm/fnm/volta):
    ```bash
    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
    nvm install node
    ```
  </Tab>
  
  <Tab title="Windows">
    **Default**: User-writable by default (no admin required).
    
    npm global prefix on Windows is typically:
    ```
    %APPDATA%\npm
    ```
    
    This is user-writable, so Gentle AI does not require elevation.
  </Tab>
</Tabs>

### Shell Detection

<Tabs>
  <Tab title="macOS / Linux">
    Reads `$SHELL` environment variable:
    ```bash
    echo $SHELL
    # /bin/bash
    # /bin/zsh
    # /usr/bin/fish
    ```
  </Tab>
  
  <Tab title="Windows">
    Defaults to PowerShell when `$SHELL` not set:
    ```powershell
    powershell
    # or
    pwsh  # PowerShell Core
    ```
  </Tab>
</Tabs>

### Archive Formats

- **macOS / Linux**: `.tar.gz` (extract with `tar -xzf`)
- **Windows**: `.zip` (extract with `Expand-Archive`)
