## Overview

The interactive TUI (Terminal User Interface) provides a guided, visual workflow for configuring Gentle AI. Built with [Bubbletea](https://github.com/charmbracelet/bubbletea) and styled with the Rose Pine theme.

## Launching the TUI

Simply run the command without arguments:

```bash
gentle-ai
```

The TUI will:
1. Auto-detect your system environment
2. Scan for installed AI agents
3. Check for available updates
4. Guide you through configuration

<Info>
The TUI runs entirely in your terminal — no browser or GUI required. Works over SSH and in tmux/screen.
</Info>

## Screen Flow

The TUI consists of 10 screens that guide you through the configuration process:

### 1. Welcome Screen

<Steps>
  <Step title="First Launch">
    The welcome screen shows:
    - Gentle AI version
    - Available updates (if any)
    - Main menu options:
      - **Configure Ecosystem** → Start configuration flow
      - **Restore Backup** → Restore a previous configuration
      - **Quit** → Exit the TUI
  </Step>
  
  <Step title="Navigation">
    Use keyboard controls:
    - `↑`/`k` or `↓`/`j` — Move cursor up/down
    - `Enter` — Select option
    - `q` or `Ctrl+C` — Quit
  </Step>
</Steps>

<Tip>
Update notifications appear as a banner at the top when new versions of Gentle AI or ecosystem components are available.
</Tip>

### 3. Agent Selection

Choose which AI agents to configure:

```text
  [ ] Claude Code
  [✓] OpenCode
  [ ] Gemini CLI
  [ ] Cursor
  [ ] VS Code Copilot

  > Continue
    Back
```

<Steps>
  <Step title="Toggle Selection">
    - `Space` — Toggle agent on/off
    - Pre-detected agents are selected by default
  </Step>
  
  <Step title="Multi-Select">
    You can configure multiple agents at once.
    
    Each agent receives:
    - Independent configuration
    - Same ecosystem components
    - Agent-specific paths and strategies
  </Step>
  
  <Step title="Continue">
    Move cursor to **Continue** and press `Enter`.
    
    Requires at least one agent selected.
  </Step>
</Steps>

### 4. Persona Selection

Choose your agent's behavior mode:

```text
  > Gentleman    Teaching-oriented mentor
    Neutral      Clean, professional tone
    Custom       Bring your own persona
```

<Tabs>
  <Tab title="Gentleman">
    **Recommended for learning**
    
    - Pushes back on bad practices
    - Explains the "why" behind decisions
    - Uses direct, confrontational language
    - Focuses on concepts over code
    - Adapts language to input (Spanish/English)
  </Tab>
  
  <Tab title="Neutral">
    **Recommended for experienced developers**
    
    - No personality or opinions
    - Factual and objective communication
    - Standard technical language
    - Executes requests without pushback
  </Tab>
  
  <Tab title="Custom">
    **Advanced users only**
    
    - Provide your own persona instructions
    - Full control over behavior
    - Requires manual configuration after install
  </Tab>
</Tabs>

### 5. Preset Selection

Choose what to install:

```text
  > Full Gentleman     All components + all skills
    Ecosystem Only     Core components + P0 skills
    Minimal            Engram + Persona + Permissions

| Preset | Components | Skills | Use Case |

| Full Gentleman | All 8 | All 11 | Recommended for most users |
| Ecosystem Only | 5 core | 9 SDD + 2 foundation | SDD workflow focused |
| Minimal | 3 essential | None | Lightweight setup |
| Custom | You pick | You pick | Advanced customization |

<Info>
The preset determines the initial component selection. You can review and adjust before installing.
</Info>

If any steps failed, the summary shows:

  - Re-run: gentle-ai install --component context7

Backup Available:
  Restore with: gentle-ai (select "Restore Backup")

Press Enter to exit.
```
</Accordion>

### 10. Backups Screen

View and restore previous configurations:

```text
Available Backups

  > 2026-03-13 12:34:56  (2 agents, 7 components)
    2026-03-12 09:15:23  (1 agent, 5 components)
    2026-03-10 14:42:01  (3 agents, 8 components)

    Back to Welcome
```

<Steps>
  <Step title="Select Backup">
    Move cursor to a backup entry and press `Enter`.
  </Step>
  
  <Step title="Restore">
    Gentle AI will:
    - Restore all agent configurations from snapshot
    - Revert component injections
    - Restore file contents
  </Step>
  
  <Step title="Verification">
    After restore, verify your agent configs are back to the snapshot state.
  </Step>
</Steps>

<Warning>
Restoring a backup **overwrites** current configurations. Make sure you want to revert before confirming.
</Warning>

## Keyboard Controls

| Key | Action |
|-----|--------|
| `↑` / `k` | Move cursor up |
| `↓` / `j` | Move cursor down |
| `Space` | Toggle selection (agent/component screens) |
| `Enter` | Confirm selection / Proceed |
| `Esc` | Go back to previous screen |
| `q` | Quit (except during installation) |
| `Ctrl+C` | Force quit (emergency exit) |

<Tip>
Vim-style navigation (`j`/`k`) works in addition to arrow keys.
</Tip>

## TUI Features

### Auto-Detection

The TUI automatically detects:
- Installed agents (checks for config directories)
- System architecture and OS
- Available package managers
- Installed dependencies (git, curl, node, npm)
- npm global prefix writability

### Smart Defaults

- Pre-selects detected agents
- Recommends "Full Gentleman" preset
- Resolves dependencies automatically
- Shows platform-specific install hints

### Progress Tracking

- Real-time step status updates
- Streaming logs during installation
- Error messages with context
- Rollback on failure

### Backup Management

- Automatic backups before every change
- Timestamped snapshots
- One-command restore
- Preserves agent configs and injected content

## Troubleshooting

<AccordionGroup>
  <Accordion title="TUI doesn't render correctly">
    **Issue**: Broken layout, missing colors, or garbled text.
    
    **Solutions**:
    - Make sure your terminal supports 256 colors
    - Try a modern terminal emulator (iTerm2, Alacritty, Windows Terminal)
    - Check `$TERM` environment variable: should be `xterm-256color` or similar
    - Resize terminal window (minimum 80x24 recommended)
  </Accordion>
  
  <Accordion title="Installation hangs or freezes">
    **Issue**: TUI stops responding during installation.
    
    **Solutions**:
    - Wait up to 2 minutes for npm installs to complete
    - Press `Ctrl+C` to force quit
    - Check logs at `~/.gentle-ai/logs/`
    - Run with CLI mode for verbose output: `gentle-ai install ...`
  </Accordion>
  
  <Accordion title="Cannot navigate with arrow keys">
    **Issue**: Arrow keys don't move cursor.
    
    **Solutions**:
    - Use vim keys instead: `j` (down) / `k` (up)
    - Check if your terminal is in application mode
    - Try running in a different terminal emulator
  </Accordion>
</AccordionGroup>
